Even with clear instructions, things can go wrong. Here are the most common issues Indian users face and how to fix them:

### "App Not Installed" Error on Android

This usually means there's a conflict with an existing version of the app. Solution: Go to **Settings → Apps**, find any existing HABET installation, uninstall it completely, then try installing the new APK again.

### APK Download Blocked by Browser

Some browsers block APK downloads by default. If Chrome shows a warning, tap **Download Anyway**. If the download is blocked entirely, try using a different browser like Firefox or the default Samsung Internet browser.

### iOS "Untrusted Developer" Error

If you see this error when opening the app, you haven't completed the trust step. Go to **Settings → General → VPN & Device Management** and trust the HABET certificate as described in Step 3 above.

### App Crashes on Launch

This is usually a compatibility issue. Check that your device meets the minimum requirements: Android 5.0+ or iOS 12.0+. If your device qualifies, try uninstalling and reinstalling the app. If the issue persists, clear your device's cache and try again.

### Slow Download Speed

If the APK is downloading slowly, switch from Wi-Fi to mobile data (or vice versa). Peak hours (7–11 PM IST) can cause congestion on some networks. Try downloading during off-peak hours for faster speeds.
